## Prerequisites
1. Node.js
    - [Node.js download](https://nodejs.org/en/)
2. verify installation
```  
node -v
npm -v
```   
<p align="right">(<a href="#top">back to top</a>)</p>

## Installation
1. Install dependencies at root
```
npm install
```

2. seed data
```
npm run seed 
```
